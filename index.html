Quickstart - Users

From CRAN.

# Not viable, we are currently working on a submission to CRAN.
install.packages("xwas")

Using the source from Github.

# This installs devtools package, if not already installed
> install.packages("devtools")
# This loads devtools
> library(devtools)

# This downloads the latest xwas production package from Github
> install_github('chiragjp/xwas')
# This downloads the latest xwas development package from Github
> install_github('nampho2/xwas')

> library(xwas)

Quickstart - Contributors

$ git clone git@github.com:nampho2/xwas.git

$ make deploy
$ R

> library(xwas)
